KeyMaster: Domina el teclado. Olvida el ratón.

Extensión de VS Code que detecta cuando usas el ratón y te enseña el atajo de teclado equivalente. Compatible con VS Code, Cursor y Windsurf.

100% Gratuito · Open Source · Bilingüe ES/EN
❤️ Apoya el proyecto: 💳 PayPal ☕ Ko-fi ⭐ GitHub

¿Eres desarrollador y todavía dependes del ratón para todo? ¿Cambias de pestaña con clic, abres archivos con el explorador, buscas con el menú? Con KeyMaster cada vez que uses el ratón en el editor, verás una notificación que te recuerda el atajo de teclado correcto. Sin castigos, solo aprendizaje progresivo.

🖱️ Editor de VS Code
Haces clic con el ratón en el editor:
[clic] → mover cursor a línea 42
KeyMaster te avisa:
⚠️ Clic detectado — Usa un atajo en su lugar:
→ Ctrl+G para ir a una línea específica
→ Ctrl+P para abrir archivos rápido
→ Ctrl+F para buscar en el archivo

¿Qué hace KeyMaster?

🖱️
Detecta clics de ratón Monitoriza el editor en tiempo real y avisa cada vez que usas el ratón en lugar del teclado.
🟡🟠🔴
3 modos de entrenamiento Soft: avisa pero permite. Strict: avisa y bloquea. Training: teclado visual completo.
📖
21+ atajos incluidos Navegación, edición, búsqueda, terminal, Git... Los más usados de VS Code listos desde el primer día.
🌍
Español e Inglés Todas las notificaciones en dos idiomas. Cambia desde la configuración con un clic.
⚡
Toggle instantáneo Activa o desactiva con Ctrl+Shift+K desde cualquier parte del editor. Sin menús.
🎯
Compatible con todo VS Code, Cursor AI y Windsurf. Windows, macOS y Linux. Usar en local y en remoto.

🛠️ Cómo instalar

1
Abre VS Code, Cursor o Windsurf
2
Busca "KeyMaster" en extensiones (Ctrl+Shift+X)
3
Instala y aparecerá KeyMaster: OFF en la barra de estado inferior izquierda
4
Haz clic en el ítem de la barra o pulsa Ctrl+Shift+K para activarlo
5
¡Listo! Cada vez que hagas clic con el ratón en el editor, verás la notificación con el atajo correcto

Atajos que aprenderás a usar

Abrir archivo Ctrl+P en lugar de buscar con el ratón
Ir a línea Ctrl+G en lugar de hacer clic en el número
Cambiar pestaña Ctrl+Tab en lugar de clic en la pestaña
Buscar en archivo Ctrl+F en lugar del menú
Terminal Ctrl+` en lugar de clic en el panel
Y 16 más... Navegación, edición, Git, búsqueda global, ajustes...

⌨️ Comandos de la extensión

Ctrl+Shift+K — Activar/Desactivar KeyMaster
Ctrl+Shift+J — Abrir teclado visual (próximamente)
KeyMaster: Cambiar modo — Soft / Strict / Training
KeyMaster: Referencia de atajos — Lista buscable completa
KeyMaster: Mostrar estadísticas — Tu progreso (próximamente)

¿Para qué sirve KeyMaster?

Si eres desarrollador y usas VS Code, Cursor o Windsurf, sabes que el ratón te hace perder tiempo. Con KeyMaster, cada vez que hagas clic con el ratón en el editor, verás una notificación que te enseña el atajo de teclado que deberías usar. Es como tener un mentor que te recuerda constantemente: "Usa Ctrl+P en lugar de buscar archivos con el ratón".

Características principales

🖱️
Detección de clics Detecta cuando usas el ratón en el editor y te muestra el atajo equivalente.
🟡🟠🔴
3 modos de entrenamiento Suave (avisa), Estricto (bloquea), Training (teclado visual).
📖
21+ atajos incluidos Navegación, edición, búsqueda, terminal, Git... Los atajos más usados de VS Code.
🌍
Español e Inglés Todas las notificaciones en dos idiomas. Cambia desde la configuración.
⚡
Toggle rápido Activa/desactiva con Ctrl+Shift+K desde cualquier lugar.
🎯
Compatible con todo VS Code, Cursor AI, Windsurf IDE. Windows, macOS, Linux.

⌨️ Atajos de la extensión

Ctrl+Shift+K — Activar/Desactivar KeyMaster
Ctrl+Shift+J — Abrir teclado visual (próximamente)
KeyMaster: Cambiar modo — Elegir entre soft, strict, training
KeyMaster: Referencia de atajos — Lista buscable de atajos

🛠️ Cómo instalar

1
Abre VS Code, Cursor o Windsurf
2
Busca "KeyMaster" en el Marketplace de extensiones (Ctrl+Shift+X)
3
Instala la extensión
4
Aparecerá KeyMaster: OFF en la barra de estado inferior izquierda. Haz clic para activarlo.
5
¡Listo! Ahora cada vez que hagas clic con el ratón en el editor, verás una notificación con el atajo que deberías usar.
